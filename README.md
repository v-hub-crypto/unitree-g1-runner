# Unitree G1 Cyber Runner — Solana P2E Endless Runner

![Preview](https://picsum.photos/id/1015/800/400)

Вирусный киберпанк-раннер с **Unitree G1** (2026 humanoid robot).  
Двойной прыжок • Звуки • 3 типа препятствий • Play.fun SDK

## 🚀 Фичи
- Двойной прыжок (пробел / тап)
- Звуки прыжка / сбора / смерти (Web Audio)
- Препятствия: наземные, низкие, летающие дроны
- Сбор SOL-кейсов (+80 points)
- Полная интеграция Play.fun (addPoints + endGame)

## Как запустить
1. Открой `index.html` в браузере
2. Зарегистрируй игру на [play.fun/dashboard](https://play.fun)
3. Вставь свой `gameId` в код
4. Деплой на Vercel → готово!

## Технологии
- Vanilla JS + Canvas
- Play.fun SDK
- Web Audio API (звуки без файлов)
